# Aplicacion de orientacion profesional - React Native

Migracion inicial desde la version web (Vite/React) a una app movil con Expo y React Native.

## Requisitos

- Node.js 18+
- npm

## Ejecutar

```bash
npm install
npm run start
```

Luego abre en:

- Android: `a`
- iOS: `i` (en macOS)
- Web: `w`

## Estructura

- `App.tsx`: enrutamiento interno entre secciones.
- `src/screens/HomeScreen.tsx`
- `src/screens/UniversidadScreen.tsx`
- `src/screens/CarreraScreen.tsx`
- `src/screens/TestVocacionalScreen.tsx`
- `src/screens/MentoriasScreen.tsx`

## Nota

El enlace de WhatsApp de mentorias esta como placeholder y debe actualizarse en `src/screens/MentoriasScreen.tsx`.
